\subsection{Prime counting function - \texorpdfstring{$\pi(x)$}{}}

Expected to have $ \frac{x}{\log{x}} $ primes within $[1, x]$. The prime counting function is asymptotic to $\frac{x}{\log x}$, by the prime number theorem.
